📘 90-Day Cybersecurity Study Plan

Authored by Siwar ALWELHAZI, Cybersecurity Engineer 🛡️

🌟 Introduction

This 90-day study plan is a comprehensive, self-paced roadmap designed to build a robust foundation in cybersecurity. Tailored for beginners and professionals, it offers curated resources and hands-on tasks covering networking, Linux, Python, security principles, SIEM, cloud, ethical hacking, and job readiness.

🎯 Goals and Audience

Goals:

  • Establish a strong foundation in cybersecurity concepts.
  • Gain hands-on experience with essential tools.
  • Prepare for CompTIA Network+ and Security+ certifications.
  • Develop skills in scripting, cloud security, and ethical hacking.
  • Build a consistent learning habit.

Audience:

  • Aspiring cybersecurity professionals.
  • IT professionals transitioning into cybersecurity.
  • Students in tech fields.
  • Self-learners.
  • DevOps engineers interested in security.
  • Curious individuals exploring the field.

📅 Daily Breakdown

Days 1–7: Networking Fundamentals (N10-009)

  • Watch Professor Messer’s N10-009 Playlist
  • Practice questions & exercises

Days 8–14: Security Principles (SY0-701)

  • Watch SY0-701 Playlists (Messer or Zerger)
  • Practice questions

Days 15–28: Linux Fundamentals

  • Linux Journey + Cisco Linux Unhatched

Days 29–42: Python Programming

  • Codecademy, HackerRank challenges, The Cyber Mentor course

Days 43–56: Traffic Analysis

  • Wireshark tutorials, TCPdump, Suricata on pfSense

Days 57–63: Version Control

  • Codecademy Git, Git Immersion, Try Git

Days 64–70: ELK Stack

  • Logz.io & Elastic tutorials

Days 71–77: Cloud Platforms

  • Choose GCP, AWS, or Azure

Days 78–84: Review

  • Revisit difficult topics + small projects

Days 85–90: Ethical Hacking

  • Hack The Box, VulnHub, The Cyber Mentor

Days 91–92: Resume

  • Build cybersecurity resume

Days 93–95: Job Search

  • Apply via LinkedIn/Indeed, interview prep
